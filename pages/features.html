Two-Factor Authentication

Two-Factor Authentication provides an extra layer of security to your voting site. Once a judge enters there Username and Password, they will be sent a access code via e-mail that they will then need to enter on the website to gain access voting. This is especially effective and combating block voting since a judge needs to have access to there own e-mail to login.
